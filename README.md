# AI DDR Report Generator

## Overview
This project builds an AI workflow to convert inspection and thermal reports into a structured DDR (Detailed Diagnostic Report).

## Features
- Extracts and processes inspection data
- Combines thermal and inspection insights
- Handles missing information
- Generates structured client-friendly report

## Approach
1. Text Extraction
2. Data Structuring
3. Rule-based AI Simulation
4. Report Generation

## Limitations
- Prototype-level image handling
- No real-time API integration

## Future Improvements
- Integrate GPT/Gemini
- Add OCR for image extraction
- Deploy as web app
